# 🚀 My Deep Learning Journey

This repository is a comprehensive log of my progress in learning Deep Learning, covering fundamental concepts and practical implementations using both **PyTorch** and **TensorFlow**.

---

## 📂 Project Structure

The repository is organized into specialized folders based on the framework and the complexity of the tasks:

### 1. [PyTorch Fundamentals](./Pytorch/)
This section focuses on the building blocks of the PyTorch framework.
* **Key Learning:** Tensor operations (creation, arithmetic), reshaping with `.view()`, NumPy interoperability, and GPU (CUDA) acceleration checks.
* **Notebook:** `pytorch_begin.ipynb`

### 2. [Fashion MNIST Classification with TensorFlow](./Tensorflow/)
A classic Computer Vision project using an Artificial Neural Network (ANN).
* **Key Learning:** Building sequential models, Flatten/Dense layers, `Adam` optimizer, and evaluating model performance on grayscale images.
* **Notebook:** `fashion_mnist_ann.ipynb`
